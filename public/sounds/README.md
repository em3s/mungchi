# BGM 파일 폴더

이 폴더에 CC0/무료 MP3 파일을 넣으면 사운드 데모 페이지에서 재생됩니다.

## 파일명 규칙

| 파일명 | 설명 | 추천 소스 |
|---|---|---|
| `bgm-action.mp3` | 게임 액션 BGM | Pixabay: "8-bit adventure" |
| `bgm-fun.mp3` | 신나는 BGM | Pixabay: "happy chiptune" |
| `bgm-chill.mp3` | 잔잔한 BGM | OpenGameArt: "peaceful" |

## 무료 음원 사이트

- **Pixabay**: https://pixabay.com/music/ (검색: chiptune, 8-bit, game music)
- **OpenGameArt**: https://opengameart.org/content/level-music (CC0)
- **Freesound**: https://freesound.org (CC0 필터 사용)

## 주의

파일이 없으면 데모 페이지에서 "파일 없음" 표시가 됩니다.
